# Global-voices-sales-performance-dashboard
This was a freelancing project where i cleaned the data in excel and converted it into actionable insights.
Here i have created 3 sheets named as "Database", "Pivot tables", "Dashboard". This excel dashboard is created using pivot tables which shows important metrics such as total sales by sales team, total earnings, monthly slicer, and few column charts which shows actionable insights. 
# This excel dashboard contains
• Total earnings and total paid calls
• Top 5 consultants sales revenue
• max, min and avg month revenue
• Total paid and unpaid calls
• enrolled courses and avg
• Total sales by sales team
• No of advertisements on advertisement channels
